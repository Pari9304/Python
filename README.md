# 🐍 Basic Python

This repository contains beginner-friendly Python programs and practice notebooks covering the fundamental concepts of Python programming. It is designed to help learners build a strong foundation in Python before moving on to advanced topics such as Data Analysis, Machine Learning, and Artificial Intelligence.

---

## 📚 Topics Covered

### 📌 Lists
Learn how to work with Python lists and perform common operations.

Topics:
- Creating lists
- Indexing and slicing
- Adding and removing elements
- Iterating through lists
- List comprehensions
- Basic list-based programs

---

### 📌 Tuples
Understand immutable data structures and tuple operations.

Topics:
- Creating tuples
- Tuple manipulation
- Tuple arithmetic operations
- Sorting tuples
- Frequency counting
- Tuple filtering
- Tuple and list conversions

---

### 📌 Strings
Explore Python string handling and built-in string methods.

Topics:
- String creation
- String indexing
- Multi-line strings
- String searching
- String manipulation
- String methods:
  - `count()`
  - `find()`
  - `split()`
  - `join()`
  - `partition()`

---

### 📌 Sets
Learn about Python sets and set operations.

Topics:
- Creating sets
- Adding and removing elements
- Union
- Intersection
- Difference
- Duplicate removal
- Membership testing

---

### 📌 Dictionaries
Work with key-value pair data structures in Python.

Topics:
- Creating dictionaries
- Accessing values
- Updating records
- Dictionary methods
- Iterating through dictionaries
- Practical examples

---

### 📌 Matplotlib
Introduction to data visualization using Matplotlib.

Topics:
- Bar Charts
- Scatter Plots
- Grouped Bar Charts
- Chart customization
- Labels and titles
- Legends
- Gridlines
